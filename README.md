# Sports-Arena
SportsArena is a multi-purpose sports booking platform designed to simplify the process of finding, booking, and managing sports facilities. The platform connects players with nearby sports venues, offering an intuitive, seamless, and interactive experience for both casual players and organized teams.

## Tools Used
- Figma
- UI/UX Design Principles
- User Research
